bing-maps-clustering
====================

To get started, do these things
1) clone directory 
2) cd into directory
3) npm install
4) bower install
5) grunt server

Overview

Showcasing Bing maps clustering script by Ricky Brundritt 

See app/scripts/main.js for code.
index.html has a single div element with id="map"
